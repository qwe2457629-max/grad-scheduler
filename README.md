# 研究所推甄 ・ 甄試時間衝突檢查

選學校 → 選系所 → 自動檢查碩士班甄試（推甄）第二階段面試／筆試是否撞期，並一鍵匯入 Google 日曆。

🔗 **線上版**：部署後填上你的網址（例如 `https://你的帳號.github.io/grad-scheduler/`）

## 功能
- 跨 9 所頂大（台、政、清、陽明交、成、興、正、中山、中央）選系所
- 自動偵測第二階段甄試日期撞期（同日同時段標紅）
- 月曆視圖，含各校報名期間與繳費截止提醒
- 一鍵下載 `.ics` 匯入 Google 日曆，或每場單獨加入

## 說明
- 內建為 **115 學年度（2025）** 各校簡章自動解析資料，供測試與了解流程之用。
- 部分學校（台大／清華／中央／中正）甄試日期簡章未列，標示為「待系所公告」。
- 正式報名請以各校 116 學年度官方簡章為準。

> 純前端靜態網站，無後端、無追蹤，勾選狀態僅存在使用者自己的瀏覽器。
